Termes du Web Glossaire (liste de définitions) HTML L’HyperText Markup Language, généralement abrégé HTML, est le format de données conçu pour représenter les pages web. C’est un langage de balisage permettant d’écrire de l’hypertexte, d’où son nom. WWW Le World Wide Web (WWW), littéralement la toile (d’araignée) mondiale, communément appelé le Web, et parfois la Toile, est un système hypertexte public fonctionnant sur Internet.
