Termes d'informatique Glossaire (liste de définitions) Protocole Dans les réseaux informatiques et les télécommunications, un protocole de communication est une spécification de plusieurs règles pour un type de communication particulier. Navigateur Un navigateur web est un logiciel conçu pour consulter et afficher le World Wide Web. Techniquement, c'est au minimum un client HTTP [HyperText Transfer Protocol]. Serveur Un serveur web est spécifiquement un serveur multi-service utilisé pour publier des sites web sur Internet ou un intranet. L'expression serveur Web désigne également le logiciel utilisé sur le serveur pour exécuter les requêtes HTTP [HyperText Transfer Protocol], le protocole de communication employé sur le World Wide Web.
